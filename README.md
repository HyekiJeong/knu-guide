## 경북대학교 병원
경북대학교에는 다음의 대학병원이 존재한다.
- [경북대학교 병원](경북대학교병원.md) (본원)
  - 주소 : 대구광역시 중구 동덕로 130
- 칠곡 경북대학교 병원 (분원)
  - 주소 : 대구광역시 북구 호국로 807
- 경북대학교 치과병원
  - 주소 : 대구광역시 중구 달구벌대로 2175
