# 1. 개요
대구광역시 중구 삼덕동2가에 위치한 경북대학교 부속 치과병원.

# 2. 상세
[경북대학교치과병원 전경](https://i.namu.wiki/i/SLdjcFZn5HBkEoRXCwKIbVsc1XmbsG9o8l_n8nxs0jD961xVHiCKbTZEDDZfW_7trh2-D4FHsqJ3QzvUkf2qhzJqjn3_xKDag5eqRlfAAmlV0nTdpcAscahfPyUBvxfpVnP7S2lGG0019s493LDqnQ.webp)
2008년 9월 치과진료처가 치과병원 건물을 새로 짓고 이사했다. 이전 당시만 해도 독립된 치과병원이 아니라 경북대학교병원 치과진료처가 별도의 건물을 사용하는 형태였으나, 2016년 6월 1일 특수법인 경북대학교치과병원이 출범하면서 별도의 법인으로 완전히 분리되어 현재 경북대학교병원과는 별개의 기관으로 분원이 아니다. 12병상[2] 운영중에 있다.

경북대학교병원과는 달리 칠곡경북대학교병원처럼 층수 4자 금기를 지키지 않는다.
